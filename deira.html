Introduction

Welcome to the Deira framework! The primary goal of this framework is to give a web developer robust use of CSS Grid, Flexbox and variables to create dynamic, mobile first websites.

Premade Layout Templates

Template One

Technology: CSS Grid

A basic 3 row template consisting of a header, main and footer. The template uses auto sizing for the header and footer to ensure their height will only ever be for the content. The main section is set to 1fr height, which will push the footer to the bottom of the page and give your main section as much content as needed.
